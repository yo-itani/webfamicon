# GEMINI.md

## プロジェクト概要
`webfamicon` は、Webブラウザ上で動作するファミコン（NES）エミュレータを実装する学習用プロジェクトです。
実装のベースとして、[Writing NES Emulator in Rust](https://bugzmanov.github.io/nes_ebook/) を参考にします。
学習目的であることを主眼に置き、機能を細かくタスク（Issue）として分割し、少しずつ段階的に実装・理解を進めていくアプローチをとります。

## 開発・実装のルール
- **学習ファースト:** コードを単にコピーするのではなく、各コンポーネント（CPU, ROM, PPUなど）の役割やエミュレータの仕組みの理解を深めることを優先します。
- **段階的な実装:** チュートリアルの章立てや機能単位で非常に細かくIssueを切り分け、一つずつ確実に対応・テストします。
- **PR駆動開発:** Issueに着手する際はブランチを切り、実装が完了したら必ず `gh` コマンド (`gh pr create` 等) を使用して Pull Request（PR）を作成し、メインブランチへマージするルールとします。
- **ブラウザでの動作:** 最終的にWebブラウザ上で動作させるため、こまめに動作確認ができる環境を構築します。

## 技術スタック
* **コア実装:** Rust (WebAssembly / `wasm-pack`経由でコンパイル)
* **フロントエンド:** Vanilla JS / HTML5 (必要最低限のVite構成など)
* **描画テクノロジー:** HTML5 Canvas API
